# Miniblox-All-Skins

A complete archive of Miniblox skins hosted on GitHub Pages.

## Direct Skin URLs

Every skin can be accessed directly using:

https://notsenpai52013-bit.github.io/Miniblox-All-Skins/SKIN_NAME.png

Example:

https://notsenpai52013-bit.github.io/Miniblox-All-Skins/alice.png

## Features

* Direct image URLs
* Free GitHub Pages hosting
* Easy access to Miniblox skins
* Community-maintained archive

## Usage

Use these URLs in:

* Discord bots
* Websites
* APIs
* Community tools
* Personal projects
