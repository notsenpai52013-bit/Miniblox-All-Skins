# Miniblox-All-Skins

A complete collection of Miniblox skins hosted through GitHub Pages.

## Direct Skin URLs

Each skin can be accessed directly using:

```text
https://notsenpai52013-bit.github.io/Miniblox-All-Skins/SKIN_NAME.png
```

Example:

```text
https://notsenpai52013-bit.github.io/Miniblox-All-Skins/alice.png
```

## Features

* Direct image URLs
* Fast GitHub Pages hosting
* Free to use
* Easy integration with bots and websites
* Community-maintained skin archive

## Usage

You can use these skin URLs in:

* Discord bots
* Websites
* APIs
* Community projects
* Miniblox tools

### HTML Example

```html
<img src="https://notsenpai52013-bit.github.io/Miniblox-All-Skins/alice.png">
```
